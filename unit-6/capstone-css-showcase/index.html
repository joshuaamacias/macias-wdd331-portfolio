Course mastery

Feature cards that show the techniques instead of just claiming them.

This section mixes Grid, Flexbox, container queries, custom properties, SVG icons, and layered visual styling in a way that still feels like one design.

Precision framing

Product cards scale from stacked to split layouts using a container query, so each component adapts to its own width instead of only the whole viewport.

Lighting that guides focus

Layered gradients, blend modes, and image filters give depth to the page instead of using flat blocks everywhere.

Layout systems that scale

The page uses Flexbox for alignment and navigation, Grid for the major sections, and grid-template-areas for the hero structure.

Accessible motion

Reveals, hover states, and the progress bar support the interface, but reduced-motion users still see everything fully visible and correctly placed.
